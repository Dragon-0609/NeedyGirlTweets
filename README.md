# Tweeter Custom Images Mod for Needy Streamer Overload
![Screenshot 1](Screenshots/Screenshot%201.jpg)

A mod that adds custom images to Ame's tweeter posts (can add to KAngel's posts too).

### Note: You must have BepInEx pre-configured and enabled with the game to use this mod.
[You can read how to configure BepInEx with the game here.](https://gist.github.com/amazeedaizee/ae0dd70cc0d842d6a83cd80451e3752e)

-----

The mod contain these arts:
* All normal *** action tweets (6/6)
* 4 dinder tweets

Other tweets' arts might be added later

## How to install
First make sure you have BepInEx. The mod is built for 1.4 version of the game, other versions were not tested.

After, open Tweets.zip and extract its content to root folder of the game (where BepInEx is located).

If it's working, it should add custom arts to the game after mentioned tweets I wrote above.
