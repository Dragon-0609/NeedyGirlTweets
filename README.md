# Tweeter Custom Images Mod for Needy Streamer Overload
![Screenshot 1](Screenshots/Screenshot%201.jpg)

A mod that adds custom images to Ame's tweeter posts (can add to KAngel's posts too).

### Note: You must have BepInEx pre-configured and enabled with the game to use this mod.
[You can read how to configure BepInEx with the game here.](https://gist.github.com/amazeedaizee/ae0dd70cc0d842d6a83cd80451e3752e)

-----

## Disclosure

This mod uses arts generated with AI. If you don't like AI, then don't install this mod
